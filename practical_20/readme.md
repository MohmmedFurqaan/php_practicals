## Practical 20 setup

1. clone the repostiory in your c drive generally inside the path ```c:\xampp\htdocs\php_workspace\practical_20```

2. Make sure the xamp is running and ```Apache``` and ```MySQL``` is enable

3. open the browser and type ```localhost/php_workspace/practical_20/pages/index.html```

4. click on add blogs to add blogs

5. click on update blogs to update blogs

6. click on delete blogs to delete blogs

7. click on view blogs to view blogs

8. Your output will be visible in the [phpMyadmin](http://localhost/phpmyadmin/)
